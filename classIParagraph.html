Detailed Description

IParagraph represents a paragraph. It is a block-level container, similar to the HTML

element. Paragraphs are usually blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but appart from text, a paragraph can also contain other elements, e.g. images (IImage), links (ILink), buttons (IButton), etc.

Warning
This documentation is incomplete. The user API for the document internal model is currently being defined and, thus, for this class, only some methods have been defined.
