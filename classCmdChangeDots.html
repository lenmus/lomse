CmdChangeDots::CmdChangeDots ( int  dots,
const string &  name = "Change dots" 
)

This command changes the number of dots in the selected notes and rests.

Parameters
dotsThe new number of dots (0..n) to put in the notes and rests+.
nameThe displayable name for the command. If not specified will default to "Change dots".

Remarks

  • If the selection is empty or does not contain notes or rests, the command will not be executed and will return a failure code.
  • After executing the command:
    • the selection will not be changed.
    • the cursor will not change its position.

Example

void CommandHandler::change_dots(int dots)
{
//change dots for current selected notes/rests
if (SpInteractor spInteractor = m_pPresenter->get_interactor(0).lock())
{
string name = gettext("Change note/rest dots");
SpInteractor->exec_command( new CmdChangeDots(dots, name) );
}
}
